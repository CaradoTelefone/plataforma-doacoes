# Requisitos Funcionais

RF01 – O sistema deve permitir cadastro de doadores.

RF02 – O sistema deve permitir cadastro de beneficiários.

RF03 – O sistema deve permitir cadastro de organizações sociais.

RF04 – O sistema deve permitir registro de itens para doação.

RF05 – O sistema deve listar itens disponíveis.

RF06 – O sistema deve permitir registro de solicitações de ajuda.

RF07 – O sistema deve atualizar status dos itens.

RF08 – O sistema deve excluir registros.
